## 📁 Installation

To begin using Shadow Proxy, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/NCCoder3/Shadow-Tabbed/
   ```
2. Navigate to the cloned directory:
   ```bash
   cd Shadow-Tabbed
   ```
3. Install Node Modules:
   ```npm
   npm install
   ```
4. Start the Proxy:
   ```npm
   npm start
   ```
